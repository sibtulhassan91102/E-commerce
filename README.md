Welcome to E-Commerce Aroura, a modern, full-stack digital storefront designed for speed, scalability, and a seamless user experience. This project serves as a comprehensive demonstration of secure transaction handling, dynamic inventory management, and responsive UI design.

🚀 Overview
This platform provides a complete end-to-end shopping experience. Users can browse products, manage their carts, and checkout securely, while administrators have access to a robust dashboard for managing the "behind-the-scenes" logistics.
